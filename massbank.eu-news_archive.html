News Archive

Update 26 November 2020: The new MassBank data release is available and deployed on MassBank Europe. The release version is 2020.11 with the DOI: 10.5281/zenodo.4289361. The whole MassBank content is available in NIST msp format for the first time! The MassBank.msp file contains all records to be used in venodors' software library search. Enjoy and report your experiences, please!

Update 27 October 2020: The new MassBank data release is available and deployed on MassBank Europe. The release version is 2020.10 with the DOI: 10.5281/zenodo.4134734. We provide now a sql dump named MassBank-2020.10.sql. This dump contains all record information to support your implementations. Visit the populate-database section of the documentation for details.

Update 20 July 2020: The MassBank developers and MassBank contributors are proud to be part of the GitHub Archive Programm. MassBank is now human code heritage as a long term and active open source community!

Update 16 March 2020: We are proud to announce the publication of new MS library related paper A European proposal for quality control and quality assurance of tandem mass spectral libraries. All related mass spectra are available HBM4EU library.

Update 09 March 2020: MassBank Europe is now on PubChem. See also on Twitter.

Update 9 September 2020: The new MassBank data release is available and deployed on MassBank Europe. The release version is 2020.09 with the DOI: 10.5281/zenodo.4020329. LCSB contributed 7299 new records.

Update 17 July 2020: The new MassBank data release is available and deployed on MassBank Europe. The release version is 2020.06 with the DOI: 10.5281/zenodo.3903207.

Update 19 February 2020: The new MassBank server system was deployed today. Enjoy and keep us updated if any error occurs. We implemented the Metabolomics Spectrum Identifier Resolver for the high resolution imaging of mass spectra. For example MassBank Record: AC000372 resolves in metalolomics-usi-visulation of AC000372.
